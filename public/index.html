  • Mid Up

    Frameworks : MongoDB, ExpressJS, AungularJS, Node.js(MEAN stack), Socket.io, Google maps API, Oauth, Passport, Jenkins, Grunt, Bower.

    Description : 친구여럿과 함깨 어디서 만날지 정하는 애플리케이션 입니다. Socket.io를 사용하여 여러명이서 동시에 채팅과 현제 장소를 공유하면 서로에게 가장 가까운 중간 지역이 계산됩니다. 구글맵에서 GUI를 사용하여 서치원의 크기를 조절가능합니다. Facebook, Twitter, Google+ 또는 이메일로 회원가입가능며, MongDB로 데이터배이스 구현. Jenkins와 Github을 사용하여 Continuous Deployment (GitHub에 push하면 서버가 자동 업데이트됨)구현 하였습니다.

  • 블로그

    Frameworks : Ghost.io, MarkDown.

    Description : JavaScript 공부를 하며 꾸준히 노트를 적어놓은 블로그 입니다. 체팅창도 있어 궁금한점이 있으면 바로 이야기할수있습니다. Ghost.io는 JavaScript만으로 백엔드와 프론트엔드 그리고 디비까지 전부 구성하였으며 WordPress보다 안전하고 빠른 성능을 자랑하며 현제 베타이지만 앞으로 성장할것을 바라볼만합니다. 또한 에디터도 마크다운으로 글을 효율적으로 쓸수있습니다. Ghost의 잠재력을 미리봐온저는 Ghostdevs.com이라는 도메인을 현제 가지고있으며 약 2년 이네로 WordPress만큼 커질것을 확신하여 현제 틈틈히 공부중에 있습니다.

  • Watchout

    Frameworks : D3.js, SVG, Express/Node, Grunt, Bower

    Description : D3라이브러리를 이용하여 SVG의 모양 및 에니매이션을 구현하였고 Collision Detection을 사용하여 아주 간단한 게임을 만들어 보았습니다. Animation Frame을 SetInterval이 아닌 D3 timer를 이용하여 부드러운 애니메이션을 구현하였습니다. 또한 d3로 svg프로퍼티를 자유자제로 변경하여 우석들의 모양과 크기가 무작위로 변경되도록 하였습니다.
